part 2: interactive patterns

tabs · accordion · modal · tooltip — all with jQuery and common css

HTML structure

Semantic markup, flexbox nav, and container patterns. Frequently you'll see header, main, footer.

<header class="page-header">...</header>
<main class="container">...</main>

CSS building blocks

Grid, card hover, sticky header, gradient text, and now tabs/accordion.

.tabs { display: flex; ... } 
.tab-btn.active { border-bottom-color: blue; }

jQuery interactions

Toggle tabs, accordion, modal, and mobile menu — all using event handling.

$('.tab-btn').click(function() {
  $(this).addClass('active')
    .siblings().removeClass('active');
});

common questions

Space-between with centered items, wrapping on small screens. Example: header with logo + nav.

Use display: flex; min-height: 100vh; flex-direction: column; on body, then margin-top: auto on footer.
